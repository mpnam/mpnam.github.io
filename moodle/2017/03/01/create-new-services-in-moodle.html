Create New Services in Moodle

Posted in:
  • Moodle

Mobile Web Services

In order to use mobile web services (similar to other kind of services), you need to enable it as follows:

     * In Settings > Site administration > Plugins > Web services > External services  
     * Check "Enable mobile web services"

You can see the API Documentation in Settings > Site administration > Plugins > Web services > API Documentation

Declare new services functions

Firstly, you need to declare your new services in .\lib\db\services.php (recommended). You can also declare your own services.php in the db sub-folder of your plugin.
